## Installation

### Requirement
- Node JS 
- Yarn or NPM
- PHP
- Composer
- MongoDB


## Backend


```
	git clone git@github.com:dionaditya/frontendtest-niagahoster
	cd backend
	composer install
	mv .env.example .env
	php artisan serve
```
Server listening on port 8000


## Frontend

```
	git clone git@github.com:dionaditya/frontendtest-niagahoster
	cd frontend 
	yarn or npm install
	yarn serve
``` 

Now you can access the frontend apps in port 8080
